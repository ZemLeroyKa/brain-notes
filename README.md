# brain-notes

Здесь будут храниться заметки по моей деятельности

# Правила ведения заметок

1. keep it simple
2. Писать заметки простым Маркдаун текстом
3. Выделить сферы вашей жизни
4. Придумать _ярлыки_ для пометки информации
5. **Писать**.
